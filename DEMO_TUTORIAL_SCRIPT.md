**Zone 7: Far South**

- Delivery Fee updates: "Delivery Fee: KES 250" (was KES 150)

## 💡 **Production Tips**

**For Video Creation:**
- Use screen recording software (ScreenFlow, OBS, Camtasia)
- Add smooth transitions between scenes
- Highlight interactive elements with cursor circles
- Use bright, clear voice
- Add background music (optional, keep it subtle)
- Subtitles helpful for accessibility
- Add text callouts for important features

**For Live Demonstration:**
- Have a pre-made order halfway through (for time savings)
- Have test payment methods ready to show
- Zoom in on important details
- Allow time for Q&A
- Have sample confirmation page ready

**For Screenshots/Documentation:**
- Create high-resolution captures
- Annotate with arrows and numbers
- Use consistent color scheme
- Add helpful captions
- Show before/after states
